# VeriCore Developer Documentation

Welcome to the VeriCore developer documentation! This documentation is designed for GitBook and provides comprehensive guides for using VeriCore in your applications.

## What is VeriCore?

VeriCore is a **neutral, reusable trust and identity core** library for Kotlin, designed to be domain-agnostic, chain-agnostic, DID-method-agnostic, and KMS-agnostic.

## Using This Documentation

### For GitBook

This documentation is structured for GitBook. To publish:

1. Import this repository into GitBook
2. GitBook will automatically detect the `SUMMARY.md` file
3. The documentation will be organized according to the summary

### Local Development

To view locally:

1. Install GitBook CLI: `npm install -g gitbook-cli`
2. Install plugins: `gitbook install`
3. Serve locally: `gitbook serve`
4. Build: `gitbook build`
